# raspberry-pi-node-playground

## Raspberry Pi Node CoffeeScript Playground

This is some play around code for my Raspberry PI.

### Mad Jaz Fury Robot

My Fury Road vehicle. 

I wired up a dual h-bridge to control two motors for bi-directional control. I also drive a mini servo to control an axe.

Raspberry Pi + WiFi running a CoffeeScript server on node. This allows me to do ReST requests to control things via a web page.
